# Instrukcja dodania czcionki Cocomat Pro

Aby użyć czcionki Cocomat Pro na stronie, należy:

1. **Zakupić licencję** czcionki Cocomat Pro z oficjalnego źródła (np. Fontspring, YouWorkForThem)

2. **Pobrać pliki czcionek** w formatach:

   - CocomatPro-Light.woff2 (lub .woff, .otf)
   - CocomatPro-Regular.woff2 (lub .woff, .otf)
   - CocomatPro-Medium.woff2 (lub .woff, .otf)

3. **Umieścić pliki czcionek** w tym folderze (`public/fonts/`)

4. **Upewnić się**, że nazwy plików odpowiadają tym w pliku `app/globals.css`:
   - CocomatPro-Light.\*
   - CocomatPro-Regular.\*
   - CocomatPro-Medium.\*

**Uwaga:** Jeśli masz inne warianty czcionki (np. Bold, ExtraLight), możesz dodać dodatkowe definicje @font-face w pliku `app/globals.css`.
